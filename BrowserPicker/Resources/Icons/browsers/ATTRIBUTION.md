# Browser Icons Attribution

SVG fallback icons are from [Simple Icons](https://simpleicons.org/) ([MIT License](https://github.com/simple-icons/simple-icons/blob/develop/LICENSE.md)).

- `chrome.svg` — Google Chrome (trademark of Google LLC)
- `firefox.svg` — Firefox (trademark of Mozilla Foundation)
- `safari.svg` — Safari (trademark of Apple Inc.)

These icons are used only to indicate which browser a link will open in. They are not used as the Browser Picker application logo.

Primary browser icons in the UI are loaded from installed `.app` bundles via `NSWorkspace` when available.
