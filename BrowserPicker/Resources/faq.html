Frequently Asked Questions

Everything you need to know about Browser Picker.

What does Browser Picker do?

Browser Picker becomes your default browser and intercepts every link. It then routes each link to the right browser and profile — either automatically using your rules, or by asking you each time.

How do I set it as the default browser?

Open the menu bar icon and choose Set as Default Browser…. macOS may ask you to confirm. Once set, all links from other apps flow through Browser Picker.

How do rules work?

A rule matches a link by URL pattern (contains, host equals, or host suffix) and sends it to a specific browser profile. Rules are checked top to bottom — the first match wins. You can reorder them by dragging.

What happens when no rule matches?

You choose the fallback in Settings → General:

  • Use menu bar selection — opens silently in your current menu bar choice.
  • Show picker — asks which browser and profile to use each time.

Why does Safari need Accessibility permission?

To open a link in a specific Safari profile, Browser Picker drives Safari's File → New … Window menu via macOS automation, which requires Accessibility access.

After granting it, you must quit and reopen the app — macOS only applies Accessibility on a fresh launch.

Why don't my Safari profiles show up?

Safari profile names are read from a protected system database. Grant Full Disk Access in System Settings, or open Safari and use Scan Safari Profiles in Settings → Browsers.

My Chrome / Firefox profiles are missing.

Make sure the browser is installed, then click Refresh Profiles from the menu bar. Firefox profile names come from your Firefox profile groups; Chrome profiles come from Chrome's local state.

The app icon looks blank.

macOS aggressively caches app icons. Quit the app and reopen it; if it persists, log out and back in to refresh the system icon cache.
